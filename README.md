## Setup

[Install Python 3.4.6](https://www.python.org/downloads/release/python-346/)

[Install pip](https://pip.pypa.io/en/stable/installing/)

Install Requests
```
pip install requests
```

Setup a Vid.Me Application

Edit the values in VidMe.py

Run VidMe.py
```
python VidMe.py
```
